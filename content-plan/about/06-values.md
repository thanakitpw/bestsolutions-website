# Section 6: Core Values

## Subtitle
ค่านิยมของเรา

## Headline
สิ่งที่เราเชื่อและยึดมั่น

## Values (4 ข้อ — concrete ไม่ใช่ buzzword)

### 1. โปร่งใส ไม่มีซ่อน
รายงานผลทุกเดือน เห็นตัวเลขจริง เข้าถึง dashboard ได้ตลอด
ไม่มีค่าใช้จ่ายแอบแฝง ไม่มี hidden fees
Proof: ลูกค้าทุกรายได้รับ KPI report ทุกวันที่ 5 ของเดือน

### 2. วัดผลได้จริง
ทุกแคมเปญต้องมี KPI ชัดเจน ถ้าไม่เห็นผลตามที่ตกลง เราปรับปรุงให้ฟรี
ไม่ทิ้งลูกค้ากลางทาง
Proof: เคยจ่ายเงินแต่ไม่เห็นผล? เราทำงานกับ KPI ที่ตกลงกันไว้ล่วงหน้า

### 3. ลงมือทำ ไม่ใช่แค่แนะนำ
เราไม่ใช่ที่ปรึกษาที่ให้คำแนะนำแล้วจบ เราลงมือทำจริง ดูแลจริง
ตั้งแต่วางแผนจนถึงส่งมอบผลลัพธ์
Proof: ทุกโปรเจกต์มี dedicated team ดูแลตลอดสัญญา

### 4. เข้าใจงบประมาณ SME
เราออกแบบ packages ที่ SME เข้าถึงได้ ไม่ผูกมัดสัญญารายปี
จ่ายเท่าที่ใช้ ยกเลิกได้ตลอด
Proof: ลูกค้ากว่า 80% อยู่กับเราเกิน 1 ปี โดยสมัครใจ
