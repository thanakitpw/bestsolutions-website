## การจัดกลุ่มบริการ (3 กลุ่ม ตาม Customer Journey)

### กลุ่ม 1: สร้างตัวตนออนไลน์ (Foundation)

| บริการ | Description | ราคา |
|---|---|---|
| รับทำเว็บไซต์ | ออกแบบและพัฒนาเว็บไซต์มืออาชีพ โหลดเร็ว รองรับมือถือ พร้อม SEO ตั้งแต่วันแรก | เริ่มต้น 15,000 บาท |
| Video Production | ผลิตวิดีโอโปรโมทสินค้าและแบรนด์ ตั้งแต่ Script จนถึง Edit พร้อมลงโซเชียล | เริ่มต้น 10,000 บาท/ชิ้น |

### กลุ่ม 2: เพิ่มยอดขายและลูกค้าใหม่ (Growth)

| บริการ | Description | ราคา |
|---|---|---|
| ยิงแอดโฆษณา | บริหาร Facebook Ads, Google Ads ให้ถึงกลุ่มเป้าหมายที่ใช่ เพิ่ม ROAS ลด Cost per Lead | เริ่มต้น 8,000 บาท/เดือน |
| รับทำ SEO | เพิ่มอันดับ Google ด้วย Content และ On-page SEO ดึงลูกค้าโดยไม่ต้องพึ่งแอด | เริ่มต้น 10,000 บาท/เดือน |
| ดูแลเพจโซเชียล | ครีเอทคอนเทนต์ ดูแลโพสต์ ตอบแชท ให้แบรนด์ดูน่าเชื่อถือทุกวัน | เริ่มต้น 12,000 บาท/เดือน |
| Lead Generation | ระบบดึง Lead คุณภาพสูงเข้าธุรกิจอย่างต่อเนื่อง ครอบคลุมทุก Channel | เริ่มต้น 20,000 บาท/เดือน |
